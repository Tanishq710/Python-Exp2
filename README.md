# Grade Calculator

A simple Python script that determines a student's division based on their percentage score.

## Description

This program prompts the user to enter a student's percentage and then classifies them into one of the following categories:
- First Division: 61-100%
- Second Division: 50-59%
- Third Division: 40-49%
- Failed: Below 40%

## Requirements

- Python 3.x installed on your system

## How to Run

1. Ensure you have Python installed.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the script: `e:\Python\exp 2`
4. Run the script using: `python "Grade calculator.py"`
5. Enter the percentage when prompted.

## Example

```
Enter Percentage of Student: 75
Student Gets First Division
```
